# Stage Monitoring Tool

## Projectbeschrijving

De Stage Monitoring Tool is een webapplicatie die het volledige stageproces centraliseert en vereenvoudigt voor studenten, docenten, mentoren, stagecommissie en administratie.

Het platform biedt één centrale omgeving waarin gebruikers stageaanvragen kunnen beheren, logboeken kunnen opvolgen, feedback kunnen uitwisselen, evaluaties kunnen raadplegen en stageovereenkomsten digitaal kunnen ondertekenen.

## Doelstelling

Het doel van deze applicatie is om de communicatie en opvolging van stages efficiënter te maken door alle relevante informatie op één plaats beschikbaar te stellen.

## Functionaliteiten

### Student
- Stageaanvraag indienen
- Logboeken invullen
- Feedback bekijken
- Evaluaties raadplegen
- Stageovereenkomst ondertekenen

### Docent
- Studenten opvolgen
- Logboeken bekijken
- Mentorfeedback raadplegen
- Evaluaties bekijken
- Eindevaluatie en eindconclusie toevoegen
- Stageovereenkomst ondertekenen

### Mentor
- Studenten begeleiden
- Feedback geven
- Evaluaties invullen
- Stageovereenkomst ondertekenen

### Stagecommissie
- Stageaanvragen goedkeuren of afkeuren
- Feedback geven op aanvragen
- Stageovereenkomsten raadplegen

### Administratie
- Documenten beheren
- Contracten uploaden
- Stagegegevens registreren
